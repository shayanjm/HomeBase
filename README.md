Installation instructions:
==========================

If using ```nvm``` please use node v 0.10.7. It fixes an issue with PhantomJS.

+ Install Karma:

```npm install -g karma```

+ Install Mocha:

```npm install -g mocha```

+ Install PhantomJS

```npm install -g phantomjs```


+ Install everything in ```package.json```

```npm install```

+ Run!

```grunt server```
